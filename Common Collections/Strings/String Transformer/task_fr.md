## Transformateur de chaîne

Construisons une petite machine sous la forme d'une fonction. En entrée, nous allons donner une liste de chaînes de caractères et de commandes. Ces commandes déterminent quelle action sera appliquée à la chaîne. Cela peut être :
 - Mettre la chaîne en majuscules
 - Tronquer la chaîne
 - Ajouter "bar" à la chaîne un certain nombre de fois

La forme exacte sera la suivante :
 - L'entrée sera un Vecteur d'un tuple de longueur 2, le premier élément est la chaîne de caractères, le second est la commande.
 - L'élément de sortie sera un Vecteur de chaînes de caractères.
