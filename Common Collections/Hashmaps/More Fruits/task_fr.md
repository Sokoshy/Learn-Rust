## Plus de fruits

Un panier de fruits sous la forme d'une table de hachage est donné. La clé représente le nom du fruit et la valeur représente combien de ce fruit particulier se trouve dans le panier. Vous devez mettre *PLUS DE 11* fruits dans le panier. Trois types de fruits - Pomme (4), Mangue (2) et Litchi (5) sont déjà présents dans le panier. Vous n'êtes pas autorisé à ajouter plus de ces fruits !

Faites en sorte que votre code réussisse les tests !

<div class="hint">Utilisez les méthodes <code>entry()</code> et <code>or_insert()</code> de <code>HashMap</code> pour y parvenir.</div>
