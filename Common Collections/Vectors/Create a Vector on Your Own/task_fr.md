## Créer un vecteur par vous-même

Votre tâche est de créer un `Vec` qui contient exactement les mêmes éléments que dans le tableau `a`.

Faites en sorte que le code compile et passe le test !

<div class="hint">En Rust, il existe deux façons de définir un vecteur.

Une façon est d'utiliser la fonction `Vec::new()` pour créer un nouveau vecteur
  et de le remplir avec la méthode `push()`.
</div></li>

<div class="hint">La deuxième façon, qui est plus simple, est d'utiliser la macro <code>vec![]</code> et
  de définir vos éléments à l'intérieur des crochets.</div></li>
   
<div class="hint">Consultez ce <a href="https://doc.rust-lang.org/stable/book/ch08-01-vectors.html">chapitre</a> du livre de Rust pour en savoir plus.</div></li>
