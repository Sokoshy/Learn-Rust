## En dehors du module

Faites-moi compiler, sans retirer la macro du module !

<div class="hint">
  Pour utiliser une macro en dehors de son module, vous devez faire quelque chose de spécial au module pour soulever la macro dans son parent.

  Le même truc fonctionne également sur les déclarations `extern crate` pour les crates qui ont exporté des macros, si vous en avez déjà vu.
</div>
