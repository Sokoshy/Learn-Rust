## Les génériques dans les vecteurs

Ce programme de liste de courses ne se compile pas !
Utilisez vos connaissances des génériques pour le corriger.

<div class="hint">Les vecteurs en Rust utilisent des génériques pour créer des tableaux dynamiquement de n'importe quel type. Vous devez indiquer au compilateur le type que nous ajoutons à ce vecteur.</div>
