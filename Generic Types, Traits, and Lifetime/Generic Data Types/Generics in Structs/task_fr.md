## Génériques dans les struct

Ce puissant conteneur permet de stocker une valeur entière positive. Réécrivez-le en utilisant des génériques afin qu'il prenne en charge l'encapsulation de n'importe quel type.

<div class="hint">Actuellement, nous encapsulons uniquement des valeurs de type <code>u32</code>.
Peut-être pourrions-nous mettre à jour les références explicites à ce type de données d'une manière ou d'une autre ?</div>

<div class="hint">
Si vous êtes encore bloqué, consultez à nouveau la sous-section "Dans les définitions de méthode" dans "Types de données génériques".</div>
