## Traits : définir un comportement commun

Un *trait* informe le compilateur Rust sur la fonctionnalité qu'un type particulier possède et peut partager avec d'autres types. Nous pouvons utiliser les traits pour définir un comportement commun de manière abstraite. Nous pouvons utiliser des bornes de trait pour spécifier qu'un générique peut être n'importe quel type ayant un certain comportement.

> Remarque : Les traits sont similaires à une fonctionnalité souvent appelée *interfaces* dans d'autres langages, bien qu'avec certaines différences.
