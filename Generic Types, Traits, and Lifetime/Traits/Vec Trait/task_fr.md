## Traité Vec

Votre tâche est d'implémenter le trait
`AppendBar` pour un vecteur de chaînes de caractères.

Pour implémenter ce trait, réfléchissez
un moment à ce que signifie 'ajouter "Bar"'
à un vecteur de chaînes de caractères.

Pas de code passe-partout cette fois,
vous pouvez le faire !

<div class="hint">Remarquez comment le trait prend possession de <code>self</code> et retourne <code>Self</code>.
Essayez de modifier le vecteur de chaînes de caractères entrant.</div>
<div class="hint">Les vecteurs fournissent des méthodes appropriées pour ajouter un élément à la fin. Consultez la documentation à : <a href="https://doc.rust-lang.org/std/vec/struct.Vec.html">https://doc.rust-lang.org/std/vec/struct.Vec.html</a></div>
