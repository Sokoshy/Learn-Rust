## Donne-lui une licence

Votre tâche consiste à implémenter le trait `Licensed` pour
les deux structures et à les faire retourner la même
information (à savoir, le texte `Some information`) sans implémenter la même fonction deux fois.

Réfléchissez à ce que vous pouvez ajouter au trait `Licensed`.

<div class="hint">
Les traits peuvent avoir une implémentation par défaut pour les fonctions. Les structures qui implémentent
le trait peuvent alors utiliser la version par défaut de ces fonctions si elles choisissent de ne pas
implémenter la fonction elles-mêmes.

En savoir plus [ici](course://Generic+Types,+Traits,+and+Lifetime/Traits/Default+Implementations).
</div>
