## Manger des en-cas

Vous pouvez amener des chemins de module dans des portées et leur donner de nouveaux noms avec les mots-clés `use` et `as`. Corrigez ces instructions `use` pour que le code se compile.

<div class="hint">
Le module <code>delicious_snacks</code> essaie de présenter une interface externe différente de sa structure interne (les modules <code>fruits</code> et <code>veggies</code> et les constantes associées). Complétez les instructions <code>use</code> pour correspondre aux utilisations dans main et trouvez le mot-clé manquant pour les deux constantes.
</div>
