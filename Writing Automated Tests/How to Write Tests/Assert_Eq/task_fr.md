Ce test a un problème — faites en sorte que le test se compile !

<div class="hint">
  Comme dans l'exercice précédent, vous n'avez pas besoin d'écrire de code pour que ce test se compile et s'exécute.
  <code>assert_eq!</code> est une macro qui prend deux arguments et les compare.

  Essayez de lui donner deux arguments qui sont différents !
  Essayez de lui donner deux valeurs de types différents !
