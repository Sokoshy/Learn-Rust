## Éditeur

L'<b>Éditeur</b> est votre terrain de jeu où vous allez programmer. Vous pouvez y expérimenter pendant que vous travaillez sur les tâches théoriques et les quiz sans être vérifié.

Pour les exercices de programmation, l'Éditeur est l'endroit où vous corrigerez le code existant ou écrirez votre propre code à partir de zéro. Ce code sera vérifié.

Pour exécuter votre code à tout moment, choisissez l'option Exécuter dans le menu contextuel ou appuyez sur <span style="color: #509DD6">&shortcut:Run;</span> :

Il peut parfois être utile d'exécuter la fonction `main()` (contenue dans le fichier `main.rs`, bien qu'il ne soit pas présent dans toutes les tâches) pour voir quel résultat produit votre code. Si vous souhaitez retourner à l'Éditeur et vous concentrer sur votre code, le moyen le plus rapide de le faire est d'utiliser la commande Masquer toutes les fenêtres (<span style="color: #509DD6">&shortcut:HideAllWindows;</span>). Pour faire réapparaître toutes les fenêtres, répétez la commande.
