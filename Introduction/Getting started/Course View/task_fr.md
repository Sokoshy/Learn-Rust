## Vue du cours

<b>Vue du cours</b> vous montre le programme du cours : une liste de leçons avec des tâches.

Vous pouvez accéder à n'importe quelle tâche en double-cliquant sur son nom.

Pour masquer la fenêtre Vue du cours, cliquez sur le bouton Fenêtre de l’outil Projet ou appuyez sur <span style="color: #509DD6">&shortcut:ActivateProjectToolWindow;</span>. Cela vous donnera plus d'espace pour les fenêtres de l'Éditeur et de la Description de la tâche.

Pour afficher de nouveau la fenêtre Vue du cours masquée, cliquez à nouveau sur le bouton Fenêtre de l’outil Projet (ou appuyez sur <span style="color: #509DD6">&shortcut:ActivateProjectToolWindow;</span>).
