## Utiliser `as`

Le transtypage en Rust se fait via l'utilisation de l'opérateur `as`.  
Veuillez noter que l'opérateur `as` n'est pas uniquement utilisé pour le transtypage.  
Il aide aussi à renommer les imports.

L'objectif est de s'assurer que la division ne provoque pas d'erreur à la compilation  
et renvoie le type approprié.

<div class="hint">Utilisez l'opérateur <code>as</code> pour convertir l'un des opérandes dans la dernière ligne de la fonction <code>average</code> en le type de retour attendu.</div>
