## Itérer sur un vecteur avec des itérateurs

Un `Vec` de nombres est donné. Votre tâche est de compléter la fonction afin que chaque nombre dans le `Vec` soit multiplié par 2.

<div class="hint">Lisez la documentation sur la fonction <code>map</code>. Pour accéder à la documentation, essayez de déplacer votre curseur de souris sur l'appel à cette fonction dans votre code.</div>
