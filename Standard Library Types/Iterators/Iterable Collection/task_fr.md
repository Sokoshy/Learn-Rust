## Collection itérable

Lors de l'exécution d'opérations sur les éléments d'une collection, les itérateurs sont essentiels.
Ce module vous aide à vous familiariser avec la structure de l'utilisation d'un itérateur et
avec le parcours des éléments au sein d'une collection itérable.

Consultez les indices si vous êtes bloqué !

<div class="hint">Nous devons appliquer quelque chose à la collection <code>my_fav_fruits</code> avant de commencer à la parcourir.
Qu'est-ce que cela pourrait être ? Regardez la définition du vecteur struct pour vous inspirer :
<a href="https://doc.rust-lang.org/std/vec/struct.Vec.html">https://doc.rust-lang.org/std/vec/struct.Vec.html</a>.</div>
