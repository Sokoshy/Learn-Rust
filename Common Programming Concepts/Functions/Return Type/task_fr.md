## Type de retour

Ce magasin fait une promotion : si le prix est un nombre pair, vous obtenez une réduction de 10 (unité monétaire), mais s'il est impair, la réduction est de 3 (unité monétaire).

<div class="hint">
  Le message d'erreur indique la ligne 6 et mentionne qu'il attend un type après le <code>-></code>.
  C'est là que le type de retour de la fonction doit être – regardez la fonction <code>is_even</code> pour un exemple !
</div>
