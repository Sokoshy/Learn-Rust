## Changer de type

Fais-moi compiler !

<div class="hint">
  Dans la liaison de variables mutables, nous avons déjà appris comment rendre une variable immuable mutable en utilisant un mot-clé spécial. Malheureusement, cela ne nous aide pas beaucoup dans cet exercice car nous voulons attribuer une valeur d'un type différent à une variable existante. Parfois, vous pouvez aussi vouloir réutiliser des noms de variables existants parce que vous convertissez simplement des valeurs en différents types comme dans cet exercice.

Heureusement, Rust a une solution puissante à ce problème : le 'masquage'!
Vous pouvez en lire plus sur le 'masquage' dans la [tâche correspondante](course://Common Programming Concepts/Variables/Shadowing).
Essayez de résoudre cet exercice ensuite en utilisant cette technique.
</div>
