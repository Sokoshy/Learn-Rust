## Type de constante

Faites-moi compiler !

<div class="hint">
  Nous connaissons les variables et la mutabilité, mais il existe un autre type important de
variable disponible : les constantes.
</div>

<div class="hint">
Les constantes sont toujours immuables et elles sont déclarées avec le mot-clé <code>const</code> plutôt
que le mot-clé <code>let</code>.
Les types de constantes doivent également toujours être annotés.
</div>

<div class="hint">

Lisez-en plus sur les constantes dans la tâche [Constantes](course://Common+Programming+Concepts/Variables/Constants).
</div>
