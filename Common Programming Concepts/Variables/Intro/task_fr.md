## Concepts de programmation communs

Dans cette section, nous aborderons des concepts de programmation communs tels que :

* Variables et mutabilité
* Types de données
* Fonctions
* Contrôle de flux

Cette section couvre des concepts qui apparaissent dans presque tous les langages de programmation et comment ils fonctionnent en Rust. De nombreux langages de programmation ont beaucoup en commun à leur cœur. Aucun des concepts présentés dans ce chapitre n'est unique à Rust, mais nous les discuterons dans le contexte de Rust et expliquerons les conventions autour de l'utilisation de ces concepts.

En particulier, vous apprendrez à propos des variables, des types de base, des fonctions, des commentaires et du contrôle de flux. Ces fondations seront présentes dans chaque programme Rust, et les apprendre tôt vous donnera un noyau solide pour débuter.

### Mots-clés

Le langage Rust possède un ensemble de mots-clés réservés à l'usage du langage uniquement, comme dans d'autres langages. Gardez à l'esprit que vous ne pouvez pas utiliser ces mots comme noms de variables ou de fonctions. La plupart des mots-clés ont des significations spéciales, et vous les utiliserez pour effectuer diverses tâches dans vos programmes Rust ; quelques-uns n'ont actuellement aucune fonctionnalité associée, mais ont été réservés pour une fonctionnalité qui pourrait être ajoutée à Rust à l'avenir. Vous pouvez trouver une liste des mots-clés dans [l'Annexe A](https://doc.rust-lang.org/stable/book/appendix-01-keywords.html).
