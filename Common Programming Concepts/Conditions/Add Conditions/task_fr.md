## Ajouter des conditions

Étape 1 : Faites en sorte que le code se compile !

Étape 2 : Faites en sorte qu'il affiche "bar" dans le cas de "fuzz" et "baz" dans le cas de tout autre chose pour réussir les tests !
Consultez les indices si vous êtes bloqué :)

<div class="hint">Pour cette première erreur de compilation, il est important en Rust que chaque bloc conditionnel retourne le même type !</div>

<div class="hint">Pour réussir les tests, vous aurez besoin de quelques conditions vérifiant différentes valeurs d'entrée.</div>
