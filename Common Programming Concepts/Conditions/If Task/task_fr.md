## Retourner le Plus Grand Nombre

Complétez cette fonction pour retourner le plus grand nombre !

Ne pas utiliser :

* `return`
* un autre appel de fonction
* des variables supplémentaires

<div class="hint">
  Il est possible de faire cela en une seule ligne si vous le souhaitez !

  Quelques exemples similaires dans d'autres langues :
  - En C(++) cela serait : `a > b ? a : b`
  - En Python cela serait :  `a if a > b else b`
</div>
<div class="hint">
  Rappelez-vous en Rust que :

  * la condition <code>if</code> n'a pas besoin d'être entourée par des parenthèses
  * les conditionnels <code>if</code> / <code>else</code> sont des expressions
  * Chaque condition est suivie d'un bloc <code>{}</code>.
</div>
