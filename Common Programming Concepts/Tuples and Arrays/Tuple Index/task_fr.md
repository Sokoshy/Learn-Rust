## Utiliser un indice de tuple

Utilisez un indice de tuple pour accéder au deuxième élément de `numbers`.

Vous pouvez mettre cela directement dans le `println!`.

<div class="hint">

  Bien que vous puissiez utiliser un destructuring avec <code>let</code> pour le tuple ici, essayez plutôt d'y accéder par index, comme expliqué dans le dernier exemple de [Le type Tuple](course://Common Programming Concepts/Tuples and Arrays/The Tuple Type).

  Vous avez maintenant un autre outil dans votre boîte à outils !
</div>
