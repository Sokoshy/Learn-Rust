## Créer un tableau

Créez un tableau avec au moins 100 éléments. Exécutez le fichier main.rs pour voir ce que votre code produit. Ne modifiez pas les lignes qui sont renvoyées &mdash; le test vérifie la chaîne de caractères `"Wow, that's a big array!"`.

<div class="hint">
  Il existe une méthode courte pour initialiser des tableaux avec une certaine taille qui ne nécessite pas de taper 100 éléments (mais vous pouvez le faire si vous le souhaitez !).

  Par exemple, vous pouvez faire :
  ```rust
  let array = ["On est bientôt arrivés ?"; 10];
  ```
</div>
