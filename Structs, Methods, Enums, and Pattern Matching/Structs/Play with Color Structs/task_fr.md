## Jouer avec les structures de couleur

Complétez tous les champs TODO pour que les tests passent !
Vous devez définir une structure classique avec deux champs : `name` et `hex` puis créer une instance de cette structure avec les valeurs **"green"** et **"#00FF00"** correspondant à ces champs.
Vous devez également définir une structure tuple avec deux champs de type chaîne de caractères et en instancier une avec les mêmes valeurs qu'auparavant.

<div class="hint">
Si vous avez des difficultés, cherchez des exemples dans cette leçon. Vous aurez besoin à la fois de structures régulières avec des champs nommés et de structures tuple.
</div>
