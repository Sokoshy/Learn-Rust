## Utiliser les structs pour structurer des données liées

Une *struct*, ou *structure*, est un type de données personnalisé qui vous permet de nommer et d'emballer ensemble plusieurs valeurs connexes qui forment un groupe significatif. Si vous êtes familier avec un langage orienté objet, une *struct* est similaire aux attributs de données d'un objet. Dans ce chapitre, nous comparerons et contrasterons les tuples avec les structs, nous montrerons comment utiliser les structs, et nous discuterons comment définir des méthodes et des fonctions associées pour spécifier le comportement associé aux données d'une struct. Les structs et les enums (abordés dans le chapitre "Enums") sont les éléments de base pour créer de nouveaux types dans le domaine de votre programme afin de tirer pleinement parti de la vérification de type au moment de la compilation de Rust.
