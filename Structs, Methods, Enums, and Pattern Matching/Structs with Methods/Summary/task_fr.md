## Résumé

Les structures vous permettent de créer des types personnalisés qui sont adaptés à votre domaine. En utilisant des structures, vous pouvez maintenir les éléments de données associés connectés entre eux et nommer chaque élément pour rendre votre code clair. Les méthodes vous permettent de spécifier le comportement que les instances de vos structures ont, et les fonctions associées vous permettent d'organiser les fonctionnalités spécifiques à votre structure sans qu'une instance soit disponible.

Mais les structures ne sont pas le seul moyen de créer des types personnalisés : tournons-nous vers la fonctionnalité d'énumération de Rust pour ajouter un autre outil à votre boîte à outils.
