## Envoyer des colis

Les structures (structs) contiennent des données, mais peuvent également avoir de la logique. Dans cet exercice, nous avons défini la structure `Package`, et nous voulons tester une certaine logique qui y est attachée. Faites en sorte que le code compile et que les tests réussissent !

<div class="hint">
N'oubliez pas d'utiliser les raccourcis d'initialisation de champ dans <code>new</code> lors de l'initialisation des champs avec des variables portant les mêmes noms.
</div>

<div class="hint">
Pour <code>is_international</code> : Qu'est-ce qui rend un colis international ? Cela semble lié aux lieux qu'il traverse, non ?
</div>

<div class="hint">
Pour <code>get_fees</code> : En général, plus c'est gros, plus c'est cher. Nous n'avons pas la taille, mais quelque chose pourrait être pertinent ici :)
</div>

Jetez un œil dans <a href="https://doc.rust-lang.org/book/ch05-03-method-syntax.html">Le Livre</a> pour en savoir plus sur les implémentations de méthodes.
</div>
