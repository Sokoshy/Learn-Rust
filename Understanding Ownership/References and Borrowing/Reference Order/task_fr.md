## ordre de référence

Faites en sorte que le code compile uniquement en réorganisant les lignes dans `main()`, mais sans en ajouter, modifier ou supprimer aucune.

<div class="hint">
Raisonnez attentivement sur la plage dans laquelle chaque référence mutable est en vogue. Est-ce utile de mettre à jour la valeur de la référente (<code>x</code>) immédiatement après que la référence mutable soit prise ? Lisez-en davantage dans <a href="course://Understanding Ownership/References and Borrowing/Mutable References">Références Mutables</a>.
</div>
