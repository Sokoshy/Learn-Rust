## Tuples : clonage ou copie

Les tuples contenant des littéraux de texte peuvent être automatiquement copiés (voir la ligne marquée avec `*`). Cela ne fonctionne pas avec les valeurs de type `String`. Modifiez le code pour le compiler en corrigeant la ligne marquée avec `**`.

<div class="hint">
L'idée est de demander explicitement à Rust de cloner le tuple.
</div>
