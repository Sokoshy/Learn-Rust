## Construire une chaîne à partir de littéraux

Définissez la variable `message` de type `String` et remplissez-la avec les parties `part1`, `part2` et `part3`.

<div class="hint">
La variable <code>message</code> doit être mutable.
</div>

<div class="hint">
Utilisez <code>String::from</code> pour initialiser le message.
</div>

<div class="hint">
Utilisez <code>String::push_str</code> pour ajouter de nouvelles parties au message.
</div>
