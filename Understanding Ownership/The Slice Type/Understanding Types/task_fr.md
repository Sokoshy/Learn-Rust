## Maîtriser l'IDE : comprendre les types

Pouvez-vous nommer les types de toutes les expressions sélectionnées ?

Si vous avez des difficultés avec certaines d'entre elles, il existe un raccourci utile **&shortcut:ExpressionTypeInfo;** dans %IDE_NAME%.

### Tâche
Remplissez les trois dernières cases de réponse avec les types des expressions sélectionnées.
